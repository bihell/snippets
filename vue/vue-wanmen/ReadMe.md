## 环境准备

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
npm install bower -g
```

## 安装 Vue

```shell script

## 查看当前 vue 的版本清单
bower info vue
## 安装好以后会生成一个 bower_components 的目录，里面会有 vue 目录
bower install vue
```

## 安装 vue-router

```Shell
bower install vue-router
```

## 安装 Bootstrap

```shell script
bower info bootstrap
bower install bootstrap#3.4.1
```

## 安装 Jquery
```shell script
bower install jquery
```
